## 概述

DataFusion 是 Apache Arrow 生态系统中的查询执行引擎，使用 Rust 实现。

## 核心特性

- 基于 Apache Arrow 内存格式
- Rust 实现，内存安全
- 可扩展的查询优化器
- 支持自定义函数和数据源
